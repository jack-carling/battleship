## About

⚠️ WORK IN PROGRESS ⚠️

Interpretation of the classic game Battleship

## Setup

```bash
# Install dependencies
npm install

# Terminal 1: Run development server for frontend
npm start

# Terminal 2: Run backend server
npm run server
```
